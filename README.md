CINEMA-SCREEN-COMPARISON

🎬 South India Cinema Screen Comparison

An interactive, data-driven web tool to visually compare the largest cinema screens across South India — including PLF, IMAX, and Dolby formats — all drawn **to scale**.

✨ Features

**📐 Scale Chart** — All screens drawn to true proportions, bottom-aligned. Includes:
  - Width & height dimension lines
  - Aspect ratio pill inside each screen
  - 6ft human silhouette for real-world scale reference
  - Hover tooltips with full specs
  - City filter (All / Chennai / Hyderabad / Sullurpeta)
  - Toggle individual screens on/off
  - 🔗 Share your exact selection via URL

**⚖️ Side-by-Side Compare** — Pick any two screens and compare specs, dimensions, area and visual size

**🎬 Movie Fit Simulator** — See how different movie formats fill each screen:
  - IMAX Filmed (1.43:1)
  - Anamorphic Scope (2.39:1)
  - Flat (1.85:1)
  - Widescreen (2.35:1)
  - Standard 16:9 (1.78:1)
  - Shows % of screen used and black bar area

**📊 Data Table** — Full specs with aspect ratio classification, area bars, and click-to-toggle

## 🛠️ Tech Stack

- Pure **HTML + CSS + JavaScript** — zero dependencies, zero frameworks
- SVG-based chart rendering
- Canvas-based movie simulator
- Google Fonts (Bebas Neue, DM Mono, Barlow Condensed)
- Hosted on **GitHub Pages**

## 📊 Data Sources

- Screen dimensions sourced from official theater announcements, press releases, and verified community reports
- IMAX Palazzo (Nexus Vijaya) dimensions are community-sourced and unverified
- BigPix and P[XL] dimensions are approximate

## 🚀 Run Locally

No build step needed. Just open `index.html` in any modern browser:

```bash
git clone https://github.com/yourusername/cinema-screens.git
cd cinema-screens
open index.html
```
